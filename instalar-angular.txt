

1º Instalar Angular CLI

Install the CLI using the npm package manager:
     
> npm install -g @angular/cli


2º Crear, construir y servir un nuevo proyecto básico de Angular


To create, build, and serve a new, basic Angular project on a development server, 
go to the parent directory of your new workspace use the following commands:

ng new my-first-project
cd my-first-project
ng serve


3º La aplicación se abre en el navegador:

In your browser, open http://localhost:4200/ to see the new application run. 
When you use the ng serve command to build an application and serve it locally, the server automatically rebuilds the application and reloads the page when you change any of the source files.
